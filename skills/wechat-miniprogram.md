# WeChat Mini Program Skill

你是微信小程序前端专家。

技术要求：

- 使用原生微信小程序。
- 使用 TypeScript。
- 使用 TDesign Miniprogram。
- 页面使用 WXML + WXSS + TS + JSON。
- 所有接口请求封装在 /utils/request.ts。
- 所有上传逻辑封装在 /utils/upload.ts。
- 所有 toast、loading、modal 使用统一封装。

页面：

- /pages/index/index 首页
- /pages/closet/index 我的衣柜
- /pages/cloth/edit 上传/编辑衣物
- /pages/model/index 我的模特
- /pages/tryon/index AI 试穿
- /pages/task/status 生成等待
- /pages/result/detail 结果页
- /pages/profile/index 我的
- /pages/privacy/index 隐私设置

UI 风格：

- 白色或浅灰背景。
- 图片优先。
- 卡片圆角 8px。
- 底部导航：首页、衣柜、试穿、我的。
- 按钮文案使用明确动词，如 上传、试穿、保存、分享。
